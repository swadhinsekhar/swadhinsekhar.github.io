Need a web based resume so that I can capture all projects that I have worked on.

## **Installation**

```shell
python3 -m venv ven
source venv/bin/activate
pip install mkdocs
pip install mkdocs-material
pip install mkdocs-awesome-pages-plugin
```

## **Local Run**

```shell
mkdocs serve -a 0.0.0.0:9000
```

## Deploy

```shell
mkdocs gh-deploy
```
